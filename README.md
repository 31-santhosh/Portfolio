# Portfolio
Designed and developed a multi-page personal portfolio using HTML, CSS, and JavaScript with a strong focus on modern UI design and theme consistency. Implemented dark/light mode support, animated cards, hover effects, and smooth page navigation to create an engaging user experience. The website features responsive layouts, reusable components, and visually polished sections that highlight skills, certifications, and projects. Built entirely without external frameworks, emphasizing clean code, performance, and maintainability.
# Frontend Highlights
Multi-page structure (Home, About, Skills, Projects, Certifications, Contact)

Responsive and mobile-friendly design

Animated cards, hover effects & smooth navigation

Light/Dark theme support

Built without external frameworks (vanilla web technologies)
# Cloud Deployment
Amazon S3 – Static website hosting

Amazon CloudFront – CDN, HTTPS & performance optimization

AWS IAM – Secure access control

Architecture:
User → CloudFront (HTTPS) → S3
