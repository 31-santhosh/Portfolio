Wifi clone detector

Designed and implemented a lightweight cybersecurity tool to de tect Evil Twin attacks in public Wi-Fi networks. The system continuously scans nearby wireless environments and analyzes SSID, BSSID, signal strength , channel behavior, and encryption inconsistencies to identify cloned access points and alert users before connection. Built using Python with Scapy / PyWiFi, the solution is platform -independent and optimized for real-time detection.
